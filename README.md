### 👨‍💻 Sobre Mim

Boas-vindas ao meu perfil! 👋 

Sou um desenvolvedor apaixonado por **dados, segurança da informação, música e interfaces intuitivas**. Minha jornada na tecnologia transita desde a criação de sistemas modulares para a **Indústria Musical** até testes rigorosos e arquiteturas de proteção em ambientes de **Segurança Ofensiva e Defensiva**. 

Gosto de atuar em todas as pontas: seja desenvolvendo um UI/UX de alta performance, cruzando dados complexos de APIs (como Spotify), ou caçando e mitigando vulnerabilidades críticas.

---

### 🛡️ Foco em Segurança: Site-Security-Test

Tenho um interesse forte na proteção da informação e pentest. Um dos grandes marcos em segurança no meu portfólio é o projeto **[Site-Security-Test](https://github.com/atlasfernands/Site-Security-Test)**:
- Uma suíte/ambiente focado em analisar vulnerabilidades web comuns, garantir blindagem de arquiteturas e testar aplicações de ponta a ponta. 
- Exploro conceitos de análise de cabeçalhos de segurança (CORS, CSP), injeções, proteção contra ataques Man-in-the-Middle e automação de testes defensivos.

---

### 🚀 Projeto Start-Up em Destaque: Insight Artístico

**Uma plataforma SaaS voltada para a Indústria Musical**, desenhada com rigor analítico e estético para empresários e gravadoras.

#### 💫 Principais Funcionalidades:
- **Painéis Dual-Axis (Recharts):** Renderização de altíssima performance para fluxos como Ouvintes Mensais e Streams Totais.
- **Motor Spotify Web API Integrado:** Extração assíncrona, em lote, das *Audio Features* num ecossistema Python + Node.
- **Zero CLS & Framer Motion:** Orquestração SPA ultrafluida.

#### 🛠️ Tech Stack:
| Backend, Linguagens & BD | Front-End & Interfaces | Infra e Deploy |
| :--- | :--- | :--- |
| **Python (Spotipy/Background)** | **Next.js 15 (App Router)** | **Vercel** (Edge Network) |
| **Supabase** (Auth / PostgreSQL) | **React 19** & Lucide | Serverless Functions |
| API REST e WebSockets | **Tailwind CSS** + Shadcn | Git & GitHub |
