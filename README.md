## 🚀 Destaque de Projeto: Insight Artístico (Inteligência Artística)

O **Insight Artístico** é minha principal plataforma SaaS no momento, rigorosamente desenvolvida para fornecer métricas profundas e análises estratégicas para nomes e produtores da indústria musical.

🛠️ **O que ele faz:**
- 🎵 **Integração de ponta a ponta com a Spotify Web API** para extração de top tracks, features de áudio (energia, valência, dançabilidade) e popularidade atreladas ao Spotify.
- 📊 **Motor de processamento backend** via Python (Spotipy/Pandas) para extração massiva de estatísticas.
- 📈 **Gráficos comparativos** com múltiplos eixos usando Recharts atrelados as conversões de CSV do Spotify Analytics.
- 🔒 **Arquitetura Serveless** com autenticação e banco de dados rodando direto no Supabase.
- ⚡ **Construído na vanguarda do Next.js 15 (App Router)** e Tailwind CSS, aderindo estritamente aos padrões design system e acessibilidade visual WCAG 2 AA.
